Patchwork blocks/Tilkkupalikat asennus (in english below)

Pelin pitäisi toimia millä tahansa android laitteella. Ei iOS tukea.

1. Lataa "Patchwork blocks" tiedosto, lataus voi viedä hetken.
(Laite saattaa pyytää lupia liittyen lataamiseen ja on täysin turvallista hyväksyä ne).

2. Kun lataus on valmis, valitse tiedosto ilmoituksista,
tai laitteen tiedostojen hallinnan lataukset kansiosta.
Tässä vaiheessa laite voi vaatia luvan ladata "tuntemattomista lähteistä"
(jälleen tämä on täysin turvallista ja eston voi asettaa takaisin päälle asennuksen loputtua).

3. Peli asentuu laitteellesi ja pikakuvake ilmestyy valikkoon/aloitusivulle.

4. Peli on valmis pelattavaksi!

Patchwork blocks installation

Game should work on any android device. No iOS support.

1. Download the "Patchwork block" file, downloading can take a moment.
(The device might ask for permissions about downloading the game, those are completely safe to approve).

2. When downloading is finished, select the file on notifications, 
or from file explorers downloads folder.
Now the device may ask to allow "third party applications" 
(again, this is completely safe and you can set the block back on when the installation is finished).

3. The game will install to your device and it's icon will appear to the menu/start page.

4. The game is ready to be played!
